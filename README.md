# Starbucks Beverage Analytics Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing Starbucks beverage data.  
It provides insights into nutritional metrics, beverage categories, and global presence, enabling better data storytelling and decision-making.

## 🎯 Key Features
- **KPIs & Metrics**: Average calories, sugar, caffeine, and protein ranges across beverage categories.  
- **Visualizations**: Bar charts, pie charts, and category distributions to highlight nutritional trends.  
- **Interactivity**: Tooltip functionality (Page 2) and drill-through for deeper analysis.  
- **Insights**: Identification of top caffeine beverages, calorie-heavy categories, and global distribution patterns.  

## 🛠️ Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions), New Measures 
- Data Modeling  
- Visualization Design
